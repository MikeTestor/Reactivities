# Start the application

## From the terminal
Start two terminals, one for the API and one for the front-end.

### Start the API
* From the Terminal cd to the API folder and type *dotnet run* to start-up the API
* the API listens to a specific port number, this port number is show in the terminal after the API is started. You can change the port number in LaunchSettings.json

### Start the front-end
* From the terminal go to the client folder and type (ng serve)
